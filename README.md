## 背景
博物馆的使用价值是为了促进社会教育、科学研究等，可是参观博物馆有不同的人群，多动症儿童的存在经常会打扰到其他参观者，而行动不便的人士参观博物馆时又不那么自由自在。所以，针对以上两类人群，博物馆需要推出一个既适合静又适合动的APP。
